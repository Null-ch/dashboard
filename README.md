
<h1 align="center">Dashboard</h1>
Данный учебный проект включает в себя php, api, chart.js

<h3>Ссылка на проект: http://neural-network-blog.site/ </h3>


Задача: Интегрировать и визуализировать данные из 1С
Способ реализации: HHTP сервис на стороне 1С, веб приложение (PHP, API, Chart.JS, MSQL для регистрации и авторизации)
